# QMK Keymaps for Voyager Keyboard

This repository contains my custom QMK keymaps for the Voyager keyboard, with both Colemak and QWERTY layouts.

## Structure

- voyager/
    - colemak/
        - keymap.c
        - config.h
    - qwerty/
        - keymap.c
        - config.h

## Files
- `keymaps.c`: Keymap definitions
- `config.h`: Keyboard configuration
